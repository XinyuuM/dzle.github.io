eNSP使用Web登录华为USG6000V防火墙控制台

拓扑图

配置Cloud

防火墙配置

端口配置

进入防火墙的G0/0/0 端口进行设置
int g0/0/0
查看端口默认配置信息
display this
IP设置（删除默认的管理接口配置）

-非必要作-删除防火墙端口默认配置

undo ip add 192.168.0.1 24

查看外部网络配置信息

外部网卡同一网段的IP地址

ip add 192.168.15.1 24

开启所有管理服务

service-manage all permit

保存配置并查看结果

退出端口配置模式
qute
退出系统配置模式
qute
保存
save

配置完成后,就可以在外部浏览器输入IP访问这个管理页面了
